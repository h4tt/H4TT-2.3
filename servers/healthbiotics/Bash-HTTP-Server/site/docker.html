Project Description

Any software at Healthbiotics runs in Docker containers. This gives us increased security because it is separated from the operating system. Also, it is easy to share software with patients using Docker.

Healthbiotics is pushing for a containerized future. Feel free to check out our public Docker images.

Project Details

  • Moving software into containers
  • Keeping customer's information secure with least access principles
  • Removing what the end user doesn't need
